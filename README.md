### Prerequisites

* npm
  ```sh

  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/EmPierrain/RTC.git
   ```
2. Build the docker container
   ```sh
   docker build -t empierrain:rtc .
   ```
3. Run the container
   ```sh
   docker compose up -d
   ```
